Inserção de imagens

O atributo src define o arquivo de imagem que será inserido no documento. Observe que o arquivo está na pasta imagens, por isso o nome do arquivo está identificado com o path (caminho) “img/universo.jpg”.

O atributo alt aparentemente não tem nenhum impacto quando visualizamos o resultado. Entretanto seu uso é essencial para usuários com deficiência visual, que utilizam leitores de tela, pois contém uma descrição sobre o que se trata a imagem.

O elemento <figure> é para ser usado em conjunto com o elemento <figcaption> . O <figcaption> é utilizado para marcar uma legenda para conteúdos inseridos com uso do elemento <figure> . Ele sempre deve vir entre o elemento <figure> , ou seja, deve constar como elementofilho do elemento <figure> . A tag <figcaption> só é válido dentre do <figure>.

Ex:

 HTML

<figure>

<img src="img/universo.jpg"  alt="universo" >

<figcaption > Universo </figcaption >

</figure>
