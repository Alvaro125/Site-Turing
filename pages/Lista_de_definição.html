Lista de definição

Listas de definição são usadas quando temos um termo e na sequência a sua definição, estilo perguntas e respostas.

As tags usadas são <dl>, <dt> e <dd>.

  • <dl> definition list - cria a lista de definição
  • <dt> difinition term - onde o termo é proposto
  • <dd> definition definition - onde o termo é definido

Ex:

 HTML

<dl >

<dt > HTML </dt >

<dd > Linguagem de marcação de texto </dd >

<dt > CSS </dt >

<dd > Linguagem para formatação de documentos </dd >

</dl >
