<div>

A tag <div></div> é utilizada para organizar uma página em blocos (Seções).

Ao final de cada div é criada automaticamente uma quebra de linha.

Ex:

 HTML

<div>

<h1>Meu título</h1>

<h2>Meu subtítulo</h2>

</div>

<div>

<p>Meu parágrafo</p>

<p>Outro parágrafo</p>

</div>

  CSS

  div{background-color:blue;}

Meu título

Meu subtítulo

Meu parágrafo

Outro parágrafo

Mas como fazer para aplicar propriedades e valores diferentes para as diversas reações da minha página?

 Uma solução é criar uma identificação própria para cada seção.

Isso pode ser feito utilizando o atributo id

Obs:Quando utilizamos o atributo id, iniciamos a nossa configuração de propriedades no CSS com o símbolo #

 HTML

<div>

<h1>Meu título</h1>

<h2>Meu subtítulo</h2>

</div>

<div id="segunda">

<p>Meu parágrafo</p>

<p>Outro parágrafo</p>

</div>

CSS

div{background-color:blue;}

#segunda{background-color:green;}

Meu título

Meu subtítulo

Meu parágrafo

Outro parágrafo

<span>

A tag <span></span> é semelhante à <div></div>

A principal diferença é que a tag <span></span> nao aplica a quebra de linha, sendo assim utilizada para configurações "inline"(na mesma linha).

Ex:

 HTML

<p> Este é um paragrafo <span id="destaque1"> muito importante </span>

e que merece <span id="destaque2"> ser destacado </span>

</p>

  CSS

 #destaque1{color:red;}

 #destaque2{color:blue;}

Este é um paragrafo muito importante e que merece ser destacado
